üöÄ QUICK START (FOR EXPERIENCED USERS)

   npm run build

2. On cPanel:
   - Create MySQL database
   - Setup Node.js app
   - Upload files

   cd /home/username/whitelabelportal
   chmod +x quick-deploy.sh
   ./quick-deploy.sh
   nano .env  # Update credentials

4. Start Application:
   - Use cPanel Node.js App Manager
   - Click "Start App"

5. Test:
   - Visit https://yourdomain.com
   - Test all features
